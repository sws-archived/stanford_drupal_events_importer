-- SUMMARY --
Author: John Bickar

The Stanford Drupal Events Importer is a Feeds importer that imports Stanford
Event nodes from other Drupal sites. Useful in combination with the Stanford
Events Export module to migrate Stanford Event content from one Drupal site
to another.


Note: The master branch is empty. Please check out the appropriate branch
(e.g., 7.x-1.x-dev) for your version of Drupal core.
